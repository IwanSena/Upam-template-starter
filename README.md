* **Universal Deployment:** Siap untuk GitHub Pages (.nojekyll) dan GitLab Pages (.gitlab-ci.yml).
* **Zero Backend:** 100% menggunakan Browser dan Git API. Tidak memerlukan database server.
* **Client-Side Rendering:** Menggunakan JavaScript untuk mengambil data secara dinamis, memberikan pengalaman SPA (Single Page Application) sederhana.
* **Responsif:** Desain mobile-first yang bersih dan minimalis.
* **Customization Ready:** Menggunakan CSS Variables untuk memudahkan perubahan warna utama.
